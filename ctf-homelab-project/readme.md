CTF Path & Lab Builder       

Overview:                                                         
CTF Path & Lab Builder is an interactive web app designed to help cybersecurity beginners choose the right Capture The Flag (CTF) challenges based on their skill level. The app evaluates the user's experience, interests, and operating system to recommend curated platforms, beginner-friendly learning paths, and safe home lab setup instructions. It also generates simple mini-challenges so users can practice key cybersecurity skills at home.
            
Features:                                      
   Skill-level selection menu to customize recommendations
                   
   Personalized CTF platform recommendations

   Beginner-friendly learning roadmaps

   Home lab setup guide (VirtualBox, WSL, Kali Linux, etc.)

   Mini practice challenges for different categories

   Safety guidelines for ethical hacking

   Simple interface designed for new learners
