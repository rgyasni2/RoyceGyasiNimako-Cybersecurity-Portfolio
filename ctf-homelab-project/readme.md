Overview:
CTF Path & Lab Builder is an interactive web app designed to help cybersecurity beginners find the right CTF challenges for their skill level. The app analyzes the user's experience, interests, and tech setup, then recommends specific CTF platforms, a custom learning roadmap, and safe home lab setup instructions.

Features:
Skill-level slider

Personalized CTF platform recommendations

Beginner-friendly learning roadmaps

Home lab setup guide (VirtualBox, WSL, Kali Linux, etc.)

Mini practice challenges for different categories

Safety guidelines for ethical hacking

Simple interface designed for new learners
